*SubtitleRedactor

(Türkçe kısım dosyanın alt taraflarında)
--------------------------------------------/ENGLISH/----------------------------------------------------

SubtitleRedactor is an AI-powered offline subtitle redaction tool that you can freely configure.

Installation:
1. Download the appropriate build file for your system. If downloaded as a ZIP, extract it.
(There aren't any other builds yet so download SubtitleRedactor folder)

2. Download the ZIP file named AI from the following drive link: "https://drive.google.com/drive/folders/1cHR25XjXTTQFPMr0Ueokjrt3upnDgZsQ" and extract it.

3. Open the extracted AI folder, then open the sub-folder also named AI. You will see files like "model.gguf, rpc-server, llama.dll...".

4. Select all files in the folder (Ctrl + A) and copy or cut them (Ctrl + C / Ctrl + X).

5. Open the SubtitleRedactor folder you downloaded in the first step, and paste the files into the "AI" folder inside.

6. You can run the program anytime by double-clicking the SubtitleRedactor.bat file (extension may not be visible in Windows).

7. You can delete this README.txt file once you are finished.


Using the Program:

1. Once the interface opens, select your language from the top.

2. Select your .ASS file using the button right below it.

3. In the box below, type the name for the file that will be saved after redaction.

4. Select a folder for the output file using the button below.

5. Operation Mode: If you are unsure about your PC's performance, "Cool Balanced CPU Usage" mode is recommended.

6. If your PC has a GPU and CUDA 13 is installed, "Use GPU" is highly recommended.

7. If your goal is only to fix spelling and minor typos, use the default command.

8. Start the redaction.

9. You can close the program once the process is complete.

--------------------------------------------/Türkçe/----------------------------------------------------

SubtitleRedactor serbestçe konfigüre edebileceğiniz AI tabanlı bir offline altyazı redakte aracıdır.

Kurulum: 
1. Sisteminize uygun build dosyasını indirin, eğer ZIP olarak indirdiyseniz ayıklayın.
(Henüz başka build yok yani SubtitleRedactor klasörünü indirin)

2. "https://drive.google.com/drive/folders/1cHR25XjXTTQFPMr0Ueokjrt3upnDgZsQ" drive linkinden AI adlı ZIP dosyasını indirin, bu ZIP dosyasını ayıklayın.

3. Ayıklanan AI dosyasını, ardından onun da içerisindeki AI dosyasını açın, "model.gguf, rpc-server, llama.dll…" gibi dosyalar göreceksiniz.

4. Klasörün içindeki bütün dosyaları "Ctrl + A, sonra da Ctrl + X" veya başka bir yöntemle kopyalayın.

5. İlk adımda indirdiğiniz ve ayıkladığınız SubtitleRedactor dosyasını açın, içindeki AI klasörüne daha önce kopyaladığınız/kestiğiniz dosyaları yapıştırın.

6. SubtitleRedactor içerisindeki SubtitleRedactor.bat (Uzantı ismi Windows'ta görünmeyebilir) dosyasına çift tıklayarak programı istediğiniz zaman çalıştırabilirsiniz.

7. İşiniz bittikten sonra README.txt dosyasını (yani bu dosyayı) silebilirsiniz.



Programı kullanma:

1. Arayüz açılınca en yukarıdan kendi dilinizi seçin.

2. Onun hemen altındaki buton ile .ASS dosyanızı seçin.

3. Onun altındaki kutucuğa seçtiğiniz dosya redakte edilince kaydedilecek dosyanın ismini yazın.

4. Onun altındaki butonu kullanarak kaydedilecek dosya için klasör seçin.

5. Çalışma türü olarak eğer bilgisayarınıza güvenmiyorsanız "Serin Dengeli CPU Kullanımı" modunu seçmeniz önerilir.

6. Eğer bilgisayarınızda GPU var ve CUDA 13 kuruluysa "GPU Kullan" seçmeniz şiddetle önerilir.

7. Eğer amacınız sadece imla ve ufak yazım hatalarını düzeltmekse varsayılan komutu kullanın.

8. Redaksiyonu başlatın.

9. İşlem tamamlanınca programı kapatabilirsiniz.
